# OTA Mini Booking Engine (Backend) By : Fikri Mobiliu

## Run

1. Create a `.env` file (optional if default values are used)
2. Create database and run `sql/schema.sql`
3. Install dependencies:

```bash
npm install
```

4. Start server:

```bash
npm run dev
```

Base API: `/api/v1`

## Env Example

```env
PORT=3000
CORS_ORIGIN=http://localhost:5173
DB_HOST=127.0.0.1
DB_PORT=3306
DB_USER=root
DB_PASSWORD=
DB_NAME=ota_booking
```
